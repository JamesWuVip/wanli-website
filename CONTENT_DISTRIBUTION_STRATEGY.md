## 📅 内容发布工作流

### 标准流程

```
1. 创作 → 在 public-site/blog/markdown/ 创建 .md 文件

2. 构建 → 运行 node build-blog.js

3. 首发 → 部署到自建博客 (zhili.wanli.ai/blog/)

4. 等待48小时 → 让搜索引擎收录原创内容
```

## 💡 内容创作技巧

### 1. 标题优化

**公式**: 数字 + 关键词 + 利益点

示例:
- ❌ "智能客服系统开发"
- ✅ "5步构建GPT-4智能客服系统，降低60%人力成本"

### 2. 开头吸引

前100字决定用户是否继续阅读：

```markdown
[痛点] 很多企业想做AI应用，但不知道从何下手。

[数据] 据统计，80%的AI项目在POC阶段就失败了。

[方案] 本文将分享我们团队在15+个AI项目中总结的实战经验...
```

### 3. 结构清晰

- 使用小标题分段
- 多用列表和表格
- 代码示例配注释
- 关键点用加粗/引用突出

### 4. 案例说明

用真实案例增强说服力：

```markdown
## 案例：某教育机构AI辅导系统

**客户背景**:
某K12在线教育机构，学生规模10万+

**痛点**:
人工答疑成本高，响应速度慢

**解决方案**:
基于GPT-4构建AI辅导助手

**效果**:
- 自动化率提升至85%
- 响应时间从5分钟降至10秒
- 成本节省60%
```
