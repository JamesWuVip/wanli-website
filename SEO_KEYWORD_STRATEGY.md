## 六、快速执行脚本

### 360搜索提交脚本

创建 `360-push.sh`:
```bash
#!/bin/bash
# 360搜索URL提交脚本

SITE="https://zhili.wanli.ai"
# 从360站长平台获取token
TOKEN="YOUR_360_TOKEN_HERE"

cat > urls-360.txt <<EOF
https://zhili.wanli.ai/
https://zhili.wanli.ai/#services
https://zhili.wanli.ai/#cases
https://zhili.wanli.ai/#about
https://zhili.wanli.ai/blog/
EOF

echo "🚀 开始推送URL到360搜索..."
# 360搜索的API地址（需要根据实际API文档调整）
curl -X POST "http://data.zz.360.cn/urls?site=$SITE&token=$TOKEN" \
  -H "Content-Type:text/plain" \
  --data-binary @urls-360.txt

rm -f urls-360.txt
echo "✅ 360搜索URL推送完成"
```

## 九、注意事项

### SEO红线（不要触碰）

❌ **黑帽SEO**:
- 关键词堆砌
- 隐藏文本
- 购买大量低质量外链
- 站群操作
- 镜像网站

❌ **技术问题**:
- 页面加载过慢（>3秒）
- 大量404错误
- 重复内容
- 移动端不友好

✅ **白帽SEO**:
- 优质原创内容
- 自然外链增长
- 技术优化
- 用户体验提升
- 合理的关键词布局
